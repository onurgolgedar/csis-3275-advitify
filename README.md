# csis-3275-advitify

## Installation

run these code after git clone

```sh
<!-- install npm packages -->
npm install

<!-- init prisma -->
npx prisma generate

<!-- run application -->
npm run dev
```

If there is no error, you can open the page: http://localhost:3000
